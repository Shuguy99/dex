## Ключевые паттерны

- **Команды** — sorted dict по длине префикса (самые длинные маскируются первыми)
- **LLM non-blocking init** — `check_available(timeout=3)` + rule-based fallback
- **Self-heal** — анализ через pylint, mypy, bandit, LLM-предложение фикса, Git-коммит
- **Hot-swap** — A/B тестирование (traffic split N:M) перед полной заменой компонента
- **Mesh** — TCP-сервер, auto-discovery, 4 уровня приватности (local→full)
- **Federated Consciousness** — propose → contribute → synthesize, 15 индикаторов приватности
